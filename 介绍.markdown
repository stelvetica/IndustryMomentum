## 二、因子设计思想与问题解决方案

### 第一类：基础动量因子（解决"如何衡量动量"）

#### 1. momentum（传统动量）

```
思想：价格惯性效应 - 过去涨的未来继续涨
公式：Momentum = P_t / P_{t-window} - 1
问题：不同行业波动率差异大，难以横向比较
```

#### 2. momentum_zscore（Z-score标准化动量）

```
思想：消除波动率差异，衡量"相对强弱的程度"
公式：z = (r - mean) / std，Factor = mean(z_{t-window+1:t})
解决：横截面标准化后，高波动和低波动行业可比
```

#### 3. momentum_rank_zscore（Rank标准化动量）

```
思想：只关注"谁比谁强"的顺序，对极端值不敏感
公式：z = (Rank - (N+1)/2) / sqrt((N+1)(N-1)/12)
解决：排名信息更稳健，避免异常值干扰
出处：东方证券《量化策略研究之六》
```

#### 4. momentum_sharpe（夏普动量）

```
思想：偏好"稳健上涨"而非"大起大落"
公式：Sharpe = mean(r) / std(r)
解决：惩罚高波动，筛选出涨得稳的行业
```

#### 5. momentum_calmar_ratio（Calmar比率）

```
思想：关注"最坏情况下的损失"
公式：Calmar = 年化收益率 / |最大回撤|
解决：偏好持续创新高、回撤小的行业，空头效果强
出处：海通证券《行业轮动系列研究7》
```

### 第二类：平稳动量因子（解决"动量反转"问题）

#### 6. momentum_turnover_adj（换手率调整动量）

```
问题：放量上涨往往是情绪过热，容易反转
思想："缩量上涨"优于"放量上涨"
公式：收益换手比 = r / Turnover，Factor = Mean / Std
解决：换手率低时信息传播慢，后续上涨概率大
出处：东方证券《量化策略研究之六》
```

#### 7. momentum_price_volume_icir（量价清洗ICIR加权）

```
问题：单一窗口动量不稳定，短期和长期信号冲突
思想：从量（成交额）和价（波动率）两个维度"清洗"动量
构造：
  - 量维度：剔除成交额最高10%交易日（去除情绪噪音）
  - 价维度：路径平滑度 = 区间总涨幅 / Σ|日收益率|
  - 合成：M = z(Factor_Amt) + z(Factor_Pric)
  - 加权：IR = Mean(IC)/Std(IC)，按IR归一化加权短期(20日)和长期(240日)
解决：动态加权适应不同市场环境
出处：长江证券《行业轮动系列(五)：动量篇》
```

#### 8. momentum_rebound_with_crowding_filter（反弹动量+拥挤度过滤）

```
问题：纯动量策略容易追高过热行业
思想：捕捉"V型反转"信号，同时规避拥挤行业
构造：
  - 传统动量：P_t / P_{t-window} - 1
  - 反弹动量：P_t / min(P_{t-window:t}) - 1（从最低点反弹）
  - 综合动量：z(传统) + z(反弹)
  - 拥挤度过滤：成交额MA在历史窗口的分位数，最高15%进入黑名单
解决：避免追高过热行业，降低均值回归风险
出处：华鑫证券《动量、拥挤度、轮动速率的统一》
```

#### 9. momentum_amplitude_cut（振幅切割动量）

```
问题：大涨大跌的日子往往是噪音
思想：只统计"风平浪静"日子的涨幅（低振幅日代表真实趋势）
构造：
  - 按振幅排序，取前70%低振幅日的对数收益率之和
  - 除以近期波动率作为风险惩罚
  - Factor = Mom_raw / Vol_recent
解决：偏好"低调稳健上涨"的行业
出处：开源证券《A股市场中如何构造动量因子？》
```

### 第三类：特质收益动量因子（解决"动量崩溃"问题）

#### 10. momentum_pure_liquidity_stripped（剥离流动性提纯动量）

```
问题：动量可能来自"非理性繁荣"（换手率异常、情绪过热）
思想：区分情绪驱动和基本面驱动的上涨
构造：
  - 计算换手率异常度和波动率异常度（时间序列Z-Score）
  - 截面OLS回归：R = α + β1*Z_Turnover + β2*Z_Vol + ε
  - 残差ε即为提纯动量（无法被流动性和情绪解释的涨幅）
解决：提取"特质动量"，剔除情绪噪音
出处：光大证券《再论动量因子》
```

#### 11. momentum_residual（行业残差动量）

```
问题：传统动量存在"动量崩溃"风险，对风格因子有高敞口
思想：先剥离风险因素的影响，再使用残差构造动量
构造：
  - 将每月行业收益率对Barra 11因子进行滚动回归（回看12个月）
  - 当月残差 = 特质动量（无法被风格因子解释的收益）
解决：如果风格因子反转，残差动量不受影响
出处：兴业证券《基于盈余惊喜、残差动量、北向资金的行业轮动模型》
核心引用：Blitz, Huij, and Martens (2011) - Residual Momentum
```

### 第四类：行业间相关性因子（解决"领先滞后"问题）

#### 12. momentum_cross_industry_lasso（Lasso行业间动量）

```
问题：行业之间存在领先滞后关系，某些行业的涨跌可以预测其他行业
思想：用Lasso回归捕捉行业间的领先滞后关系
构造：
  - 自变量：所有行业"上月"的超额收益率
  - 因变量：目标行业"下月"的超额收益率
  - 对每个行业单独做Lasso时间序列回归，用AIC选择正则化强度
  - 代入当月最新数据得到下月预测值
解决：利用行业间信息传递的时滞获取超额收益
出处：东方证券《量化策略研究之六》
```

---

### 第五类：行业内关系因子（解决"虚胖vs实壮"问题）

#### 13. momentum_industry_component（行业成分股动量）

```
问题：行业指数上涨可能仅由一两只龙头拉动（虚胖）
思想："一荣俱荣"才是真景气
构造：
  - μ_i：行业内所有成分股累计收益率的均值
  - σ_i：行业内成分股收益率的截面标准差
  - Factor = μ / σ（涨得又好又稳）
解决：筛选出行业内部共识强、反转风险小的行业
出处：东方证券《量化策略研究之六》
```

#### 14. momentum_pca（PcaMom集中度分析）

```
问题：传统动量在"快速轮动"和"抱团瓦解"行情中失效
思想：不仅看涨幅（量），更看涨得健不健康（质）
构造：
  - PcaScore：行业内代表性股票收益率矩阵的PCA分解，取前2个主成分解释率之和
  - Ratio = PcaScore_t / PcaScore_{t-Lag}（结构是凝聚了还是散了）
  - PcaMom = Mom × Ratio
信号解读：
  - 正动量 + Ratio>1（凝聚）：主升浪，Strong Buy
  - 正动量 + Ratio<1（分歧）：强弩之末，Neutral/Sell
解决：识别"共振"（真主线）vs"分歧"（鱼尾行情）
出处：兴业证券《分歧和共振——直击行业轮动痛点》
```

#### 15. momentum_lead_lag_enhanced（龙头领先增强动量）

```
问题：动量不仅要看"涨了多少"，还要看"谁带头涨的"
思想：龙头跟随效应 - 大市值龙头对基本面转好反应更快
构造：
  - Lead_sector = Demean(mean(Leader) - mean(Follower)) / (std × Volume)
  - AugMomentum = |Lead_sector| × MonthlyReturn_sector
  - Leader：前50%市值的龙头股
  - Follower：后50%市值的跟随股
解决：
  - 龙头先涨、跟随未动 → 预示"补涨"动力
  - 小盘乱炒或大小齐涨 → 后续动量较弱
出处：光大证券《行业轮动：从动量谈起》
```

## 四、核心设计理念总结

| 问题类型           | 具体问题             | 解决方案              | 代表因子                      |
| ------------------ | -------------------- | --------------------- | ----------------------------- |
| **衡量问题** | 不同行业波动率差异大 | 横截面标准化/排名     | zscore, rank_zscore           |
| **反转风险** | 放量上涨容易反转     | 换手率调整/拥挤度过滤 | turnover_adj, crowding_filter |
| **反转风险** | 大涨大跌是噪音       | 振幅切割/路径平滑度   | amplitude_cut, price_volume   |
| **崩溃风险** | 风格因子反转         | 剥离风格因子残差      | residual, liquidity_stripped  |
| **信息传递** | 行业间领先滞后       | Lasso回归预测         | cross_industry_lasso          |
| **结构问题** | 龙头拉动vs普涨       | 成分股离散度/PCA      | industry_component, pca       |
| **结构问题** | 谁带头涨             | 龙头跟随效应          | lead_lag_enhanced             |

这套因子体系从**基础衡量→风险控制→结构分析**三个层次，系统性地解决了行业动量策略面临的各种问题。
